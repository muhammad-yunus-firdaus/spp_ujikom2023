# Sistem Pembayaran SPP Online

## Tentang Proyek

Proyek ini merupakan **Ujian Kompetensi Keahlian (UKK) Tahun 2023** yang diselesaikan sebagai syarat kelulusan SMK pada kompetensi keahlian **Rekayasa Perangkat Lunak (RPL)**.

**Guru Penguji:** Bapak Maulana Firdaus, S.Pd

---

## Fitur Utama

### Login Multi-Level
- Login Siswa (menggunakan NISN)
- Login Petugas
- Login Admin

### Manajemen Data (Admin)
- Data Siswa
- Data Petugas
- Data Kelas
- Data SPP

### Transaksi Pembayaran
- Input pembayaran SPP bulanan
- Validasi pembayaran ganda
- History pembayaran

### Dashboard
- Statistik pembayaran
- Jumlah siswa
- Total pemasukan

---

## Teknologi

- PHP
- MySQL
- Bootstrap 4
- SB Admin 2
